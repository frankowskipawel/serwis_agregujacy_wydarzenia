API


Get all events
/events/all method: GET parameters: no
Get future events
/events/future method: GET parameters: no
Get past events
/events/past method: GET parameters: no
Get ongoing events
/events/ongoing method: GET parameters: no
Get event by id
/events/{id} method: GET parameters: int id
Edit update
/events/update method: PUT parameters: "id" - required, "title", "city", "startDate", "endDate", "description", "yourEmail", "yourPassword"
Add new event
/events/newEvent method: POST parameters: "title", "city", "startDate", "endDate", "description", "yourEmail", "yourPassword"
Delete event
/events/deleteEvent method: DELETE parameters: "id", "yourEmail", "yourPassword"
